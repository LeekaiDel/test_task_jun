###### Сборка на Linux:
```cmake
cmake -B build -S . && cmake --build build
```
Появившиеся исполняемые файлы:
  * my_prog - Сам файл, который делает всю работу по заданию.
  * test_reader - Тестовая программа для чтения и отображения сериализованных данных для проверки (Нужно в качество аргумента передать путь к файлу outlet.out).
